Tank Fight

Description:

Tank Fight is a both local, lan, (and online) game that is based off the original tank trouble game. Tank fight is trying to be better than tank trouble in both performance and features as the original was created in flash (which died) and tank fight is being developed in Unity using c# and the networking is being done with Mirror Networking. Also, Tank Fight will be released on itch.io when the game is complete at https://awesomegamergame.itch.io/tank-fight
