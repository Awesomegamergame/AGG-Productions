Tank Fight Demo

Description:

Tank Fight is a unity game where you and your friends are tanks battling it out in a box maze with power ups and your bullets and the last tank standing wins

Game is going to be released on april 1st.

UI Images for the menus were created by Tamagoaway#0718 on discord

The demo of the game has some limitations which are listed below:
1. No online or lan multiplayer gameplay
2. You can only play 10 rounds before being sent back to the main menu
3. You only get 1 type of power-up per game
4. The game is limited to only 2 players using wasd and arrow keys

You can buy the full game here on our itch.io page at https://awesomegamergame.itch.io/tank-fight
